<embed>  ظرفی را برای یک برنامه خارجی تعریف می کند
type => نوع فایل      src => ادرس فایل        width =>  عرض ظرف    height =>  طول ظرف


<fieldset>   عناصر مرتبط را در یک فرم گروه بندی می کند


<figcaption>   عنوان تعریف میکند<figure> برای تگ 


<figure>  محتوای خودمختار را مشخص می کند


<font>    نوع ,رنگ و اندازه متن را مشخص میکند
 font size => اندازه متن          font-family=> نوع نوشتار متن      color=> رنگ


<footer>  پانویس را برای یک سند یا بخش مشخص می کند


<form>  فرم را برای ورودی کاربر تعریف می کند 


<frame>   یک پنجره را در یک مجموعه فریم تعریف می کند
src => ادرس فریم          title =>  سرتیتر فریم


<frameset>  مجموعه ای از فریم ها را تعریف می کند


<h1> تا <h6>  عنوان را تعریف میکند


<head> حاوی اطلاعات سند است


<header> عنوان یک سند یا بخش را تعریف می کند


<hr>  توسط خط تغییر موضوعی در محتوا را تعریف می کند


<html>  ریشه سند را تعریف میکند


<i> بخشی از متن را با حالتی متفاوت تعریف می کند


<iframe>  یک قاب درون خطی را تعریف می کند
src => ادرس فریم          title =>  سرتیتر فریم


<img> تصویری را تعریف می کند
src => ادرس تصویر   alt=> توضیحات تصویر  width => عرض تصویر  height=> طول تصویر


<input>  یک ورودی را تعریف می کند
type => نوع ورودی     id => شناسه ورودی    


<ins>  متنی را که در سندی درج شده است تعریف می کند


<kbd>  ورودی صفحه کلید را تعریف می کند


<lable> عنوانی برای ورودی تعریف میکند


<legend> تعریف می کند <fieldset> عنوانی برای


<li>  یک مورد لیست را تعریف می کند


<link>  رابطه بنوعین یک سند و یک منبع خارجی را تعریف می کند 
rel =>  نوع منبع   href => ادرس منبع

<main> محتوای اصلی یک سند را مشخص می کند


<map>نقشه تصویری را تعریف می کند


<mark> متن مشخص شده را تعریف می کند


<meta> فراداده مربوط به یک سند را تعریف می کند
charset => کاراکتر  content => محتوا


<meter> اندازه گیری اسکالر را در یک محدوده شناخته شده تعریف می کند
ip => شناسه     value => مقدار    min => حداقل مقدار  max => حداکثر مقدار  


<nav> پیوندهای ناوبری را تعریف می کند


<noframes>  محتوای جایگزین را برای کاربرانی تعریف می کند که از فریم پشتیبانی نمی کنند
src => ادرس فریم 


<noscript>محتوای جایگزین را برای کاربرانی تعریف می کند که از جاوا اسکریپت پشتیبانی نمی کنند


<object> ظرفی را برای یک برنامه خارجی تعریف می کند
 width =>  عرض ظرف    height =>  طول ظرف    data =>  محتوا مورد نظر


<ol>  لیست مرتب شده ای را تعریف می کند
<li> => لیست


<optgroup>  یک گروه از گزینه های مرتبط را در یک لیست کشویی بوجود می اورد


<option>یک گزینه را در یک لیست  کشویی تعریف می کند 


<output>نتیجه یک محاسبه را تعریف می کند
name => اسم محاسبه بدست امده     for => عملیات محاسبه


<p> یک متن را تعریف می کند


<param>یک پارامتر برای یک شی تعریف می کند
name => اسم شی    value => پارامتر


<picture>ظرفی را برای چندین منبع تصویر تعریف می کند
source media => منبع و محدودیت های ظرف   srcset => ادرس تصاویر
